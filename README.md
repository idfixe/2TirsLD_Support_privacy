# 2TirsLD Support & Privacy

This repository hosts the public support and privacy pages for the 2TirsLD app.

- Support URL (App Store): `https://idfixe.github.io/2TirsLD_Support_privacy/support.html`
- Privacy URL (App Store): `https://idfixe.github.io/2TirsLD_Support_privacy/privacy.html`
- Landing page: `https://idfixe.github.io/2TirsLD_Support_privacy/index.html`

Multilingual HTML pages are generated under `l10n/<locale>/` for all app locales.

To regenerate:

```bash
python3 tools/generate_pages.py
```
