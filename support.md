# Support 2TirsLD

## Contact

Pour toute question, bug ou demande d'amelioration:

1. Ouvrir une issue GitHub sur le depot public.
2. Decrire l'appareil, la version de l'app, et les etapes pour reproduire.
3. Ajouter une capture d'ecran si possible.

## Informations utiles a fournir

- Version de 2TirsLD
- macOS / iOS / iPadOS
- Contexte (arme, rechargement, distance)
- Message d'erreur exact

## Delai de reponse

Les demandes sont traitees au fil de l'eau. Les bugs bloquants sont prioritaires.

## Support (EN)

For bug reports or feature requests, open a GitHub issue with:

- app version
- OS version
- reproduction steps
- screenshot/logs when available
